## AUDIO PRODUCTION NOTES

1. **Tone:** Professional news anchor delivery. Detached. Matter-of-fact. Slightly ominous undertones.

2. **Pacing:** Moderate pace. Allow weight to settle on consequence revelations.

3. **Duration:** 15-30 seconds per narration. Breaking news can run longer (up to 45 seconds).

4. **Ambient:** Subtle static/broadcast interference recommended for atmosphere.

5. **Dynamic Content:** Scripts marked with [If X approved/denied] require conditional audio files or text-only display with standard anchor background loop.
